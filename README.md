# GPRIndex_Analysis

## Project Overview

This project investigates whether geopolitical risk (GPR) predicts gold returns, and whether that predictive relationship is specific to war/high-risk periods rather than present at all times. The analysis uses the Caldara-Iacoviello Geopolitical Risk Index alongside gold prices, S&P 500 returns, and the VIX, over the period January 1991 to December 2024.

The empirical strategy tests four things:
- Whether GPR predicts next-month gold returns, and whether that effect strengthens in a war/high-risk sub-sample and collapses in a calm control group
- Whether geopolitical **acts** (GPRA) or geopolitical **threats** (GPRT) drive the effect (horse race)
- Whether GPR Granger-causes gold returns (and not the reverse)
- Whether GPR also affects gold return volatility (GARCH-M) and the time-varying correlation between gold and equities (DCC-GARCH)

## Script Descriptions

**`01_fred_data.py`**
Downloads the CBOE Volatility Index (VIX) from FRED, gold spot prices from a public GitHub dataset, and S&P 500 prices via yfinance. Resamples all series to monthly frequency, computes nominal log returns for gold and the S&P 500, and flags the three war episode windows (Gulf War, 9/11, Russia-Ukraine). Covers January 1991 to December 2024. Outputs `data/fred_monthly_data.csv` and a summary plot.

**`02_gpr_data.py`**
Downloads the Caldara-Iacoviello Geopolitical Risk Index (GPR, GPRA "acts", and GPRT "threats") directly from matteoiacoviello.com. Computes log-differenced GPR variables (the actual regressors used in the regressions), flags the same three war episodes, and builds the war/high-risk sub-sample (GPR > 120 or active episode) and control group (GPR < 80, no active episode). Outputs `data/gpr_monthly_data.csv` and an annotated GPR plot.

**`03_merge_and_clean.py`**
Merges the FRED dataset and the GPR dataset into a single monthly panel, creates the key dependent variable `R_GOLD_LEAD` (next month's gold return, since the analysis predicts forward), and re-confirms the war/control sub-sample flags. Runs descriptive statistics, unit root tests (ADF, PP, KPSS), ARCH effect tests, and computes the GPRA/GPRT ratio across historical episodes. Outputs the final merged panel plus war and control sub-sample CSVs.

**`04_regressions.py`**
Runs the core empirical tests of the paper: (A) OLS regressions of GPR on next-month gold returns across the full sample, war sub-sample, and control group, testing whether the effect is war-specific; (B) a horse race between GPRA and GPRT to see which drives the result, with a Wald test; (C) Granger causality tests in both directions (GPR → gold and gold → GPR); and (D) a GARCH-M model testing whether GPR also affects gold return volatility. All regressions use Newey-West HAC standard errors.

**`05_dcc_garch.py`**
Estimates a Dynamic Conditional Correlation (DCC) GARCH model between gold and S&P 500 returns, with GPR entering the mean equation. The key output is the time-varying correlation between gold and equities — a significant negative relationship with GPR indicates gold becomes a better equity hedge during high-risk periods. Also computes implied hedge ratios and optimal portfolio weights for gold.

**`06_robustness_checks.py`**
Re-tests the main OLS result under three alternative specifications: (RC1) a tercile-based GPR threshold instead of the 120/80 cutoffs, (RC2) contemporaneous rather than next-month gold returns, and (RC3) leave-one-episode-out re-estimation, dropping each war episode in turn to check the result isn't driven by any single event. Results are reported as estimated, without asserting a "robust/not robust" verdict.

## Data Sources

| Source | Data | Access |
|--------|------|--------|
| FRED (Federal Reserve Economic Data) | VIX (CBOE Volatility Index) | Free API key required |
| Caldara-Iacoviello GPR Index (matteoiacoviello.com) | GPR, GPRA (acts), GPRT (threats) | Public, no key required |
| GitHub public dataset (datasets/gold-prices) | Gold spot price (USD/troy oz) | Public, no key required |
| Yahoo Finance (via yfinance) | S&P 500 index (^GSPC) | Public, no key required |

Sample period: January 1991 to December 2024, extended back from 1993 to capture the Gulf War episode (Jan-Mar 1991) as one of the three war episodes studied, alongside 9/11 (Sep-Nov 2001) and the Russia-Ukraine invasion (Feb-Apr 2022).

## Methodology Summary

- **Dependent variable:** next month's nominal gold log return (`R_GOLD_LEAD`)
- **Key regressors:** log-differenced GPR, GPRA, and GPRT
- **Controls:** S&P 500 returns and VIX only
- **Standard errors:** Newey-West HAC (12 lags) throughout
- **Sub-samples:** War/high-risk (GPR > 120 or active episode) vs. Control (GPR < 80, no active episode)
- **Robustness checks:** tercile-based thresholds, contemporaneous vs. predictive specification, and leave-one-episode-out re-estimation

## How to Run

1. Clone the repository
2. Install dependencies:
   ```
   pip install fredapi yfinance pandas numpy matplotlib requests xlrd openpyxl scipy statsmodels arch
   ```
3. Run the scripts in order:
   ```
   python 01_fred_data.py
   python 02_gpr_data.py
   python 03_merge_and_clean.py
   python 04_regressions.py
   python 05_dcc_garch.py
   python 06_robustness_checks.py
   ```
4. Outputs are written to `data/` (processed datasets) and `results/` (tables, summaries, figures)
